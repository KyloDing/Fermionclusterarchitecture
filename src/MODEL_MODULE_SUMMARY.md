## 🔄 业务流程

### 模型导入流程
```
填写表单 → 选择文件 → 分片上传 → 创建记录 → 自动生成v1 → 完成
```

### 版本添加流程
```
选择模型 → 建议版本号 → 上传文件 → 创建版本 → 更新最新版本 → 完成
```

### 发起训练流程
```
选择版本 → 加载数据集 → 选择数据集 → 生成任务名 → 创建任务 → 跳转
```
