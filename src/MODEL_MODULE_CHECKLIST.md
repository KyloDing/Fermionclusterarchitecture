## 📝 测试场景

### 场景1：导入新模型
1. 点击"导入模型"按钮
2. 填写表单（名称：Test-Model，类型：文本）
3. 拖拽或选择文件
4. 观察上传进度
5. 验证模型出现在列表中

### 场景2：添加模型版本
1. 在列表中点击某个模型的"新增版本"
2. 查看建议的版本号
3. 上传新文件
4. 验证版本添加成功

### 场景3：查看模型详情
1. 点击模型的"查看详情"
2. 查看基本信息
3. 浏览版本列表
4. 测试各个版本操作

### 场景4：发起训练任务
1. 在详情页选择某个版本
2. 点击"发起训练"
3. 选择数据集
4. 查看数据集详情
5. 提交任务

### 场景5：删除操作
1. 尝试删除有多个版本的模型
2. 验证二次确认
3. 尝试删除版本（最后一个版本应禁用）
4. 验证删除成功
