# 政府算力券管理系统使用指南

## 功能概述

政府算力券管理系统是费米集群平台的重要组成部分,用于管理政府提供的算力补贴券,帮助用户降低算力使用成本。

## 功能特点

### 1. 算力券管理
- **我的算力券**: 查看所有已获得的算力券
  - 显示算力券代码、项目名称、发放单位
  - 实时展示使用进度和剩余额度
  - 支持按状态、类别筛选
  - 支持搜索功能

- **算力券类别**:
  - 🔴 国家级: 国家部委发放的算力券
  - 🔵 省级: 省级政府部门发放
  - 🟢 市级: 市级政府部门发放
  - 🟣 专项: 特定领域专项补贴

### 2. 申请流程
- **可申请项目**: 浏览所有开放申请的政府算力券项目
  - 查看项目详情、申请要求、截止时间
  - 了解最高补贴金额和适用范围
  - 下载申请表和相关文档

- **在线申请**:
  - 填写单位信息和项目信息
  - 说明预期使用场景和计划
  - 上传必要的申请材料
  - 提交后进入审核流程

### 3. 申请记录
- 查看所有算力券申请的历史记录
- 实时跟踪申请状态:
  - 📝 草稿: 未提交的申请
  - 📤 已提交: 已提交待审核
  - ⏳ 审核中: 正在审核中
  - ✅ 已批准: 审核通过,算力券已发放
  - ❌ 已拒绝: 审核未通过,可查看拒绝原因

### 4. 使用记录
- 详细的算力券使用历史
- 包含:
  - 使用金额和使用时间
  - 关联的订单号
  - 具体的资源类型
  - 使用场景描述

## 统计数据

系统提供以下统计指标:

1. **有效算力券**: 当前可用的算力券数量
2. **总补贴额度**: 所有算力券的总金额
3. **可用余额**: 尚未使用的算力券金额
4. **已使用金额**: 累计使用的算力券金额

## 使用场景

### 场景1: 申请国家算力补贴
1. 进入"可申请项目"标签页
2. 找到"2024年度国家算力补贴计划"
3. 点击"立即申请"按钮
4. 填写完整的申请信息:
   - 单位名称: 费米科技有限公司
   - 项目名称: 大规模语言模型训练项目
   - 申请金额: 100,000元
   - 项目描述: 基于Transformer架构的中文大语言模型预训练和微调研究
   - 预期使用: 使用8卡A100 GPU进行模型训练,预计运行3个月
5. 上传相关证明材料
6. 提交申请

### 场景2: 查看算力券使用情况
1. 进入"我的算力券"标签页
2. 找到目标算力券
3. 查看使用进度条和剩余额度
4. 点击"详情"按钮查看完整信息

### 场景3: 追踪申请进度
1. 进入"申请记录"标签页
2. 查看所有申请的状态
3. 对于"已拒绝"的申请,查看拒绝原因
4. 对于"已批准"的申请,算力券会自动发放到账户

## 算力券使用规则

### 适用范围
- ✅ AI训练任务
- ✅ 模型推理服务
- ✅ 数据处理任务
- ✅ 科研计算
- ❌ 不可转让给他人
- ❌ 仅限指定用途使用

### 使用限制
1. 算力券有有效期限制,过期自动失效
2. 必须在适用范围内使用
3. 部分算力券可能有单次使用上限
4. 算力券不可提现

### 优先级
在订单支付时,系统会优先使用算力券抵扣:
1. 先使用即将过期的算力券
2. 再使用普通算力券
3. 最后使用账户余额

## 常见问题

### Q: 如何申请政府算力券?
A: 进入"可申请项目"标签页,选择符合条件的项目,填写完整申请信息后提交。审核通过后算力券会自动发放。

### Q: 申请需要多长时间?
A: 通常需要5-10个工作日,具体时间取决于审核机构的工作安排。

### Q: 算力券可以叠加使用吗?
A: 可以。如果有多张算力券,系统会自动按优先级顺序使用。

### Q: 算力券过期了怎么办?
A: 过期的算力券无法使用。建议及时使用,避免浪费补贴额度。

### Q: 算力券使用后可以退款吗?
A: 不可以。算力券一旦使用即无法退回。

## 技术说明

### 数据模型

#### Voucher (算力券)
```typescript
interface Voucher {
  id: string;
  voucherCode: string;           // 算力券代码
  programName: string;            // 项目名称
  issuer: string;                 // 发放单位
  category: string;               // 类别: national/provincial/municipal/special
  totalAmount: number;            // 总额度
  usedAmount: number;             // 已使用金额
  remainingAmount: number;        // 剩余金额
  startDate: string;              // 开始日期
  endDate: string;                // 结束日期
  status: string;                 // 状态: active/expired/used/frozen
  applicableScopes: string[];     // 适用范围
  restrictions: string;           // 使用限制
}
```

#### VoucherApplication (申请记录)
```typescript
interface VoucherApplication {
  id: string;
  programName: string;            // 项目名称
  programCode: string;            // 项目代码
  requestedAmount: number;        // 申请金额
  projectName: string;            // 项目名称
  projectDescription: string;     // 项目描述
  organization: string;           // 申请单位
  applicant: string;              // 申请人
  status: string;                 // 状态: draft/submitted/reviewing/approved/rejected
  submitDate?: string;            // 提交日期
  reviewDate?: string;            // 审核日期
  rejectReason?: string;          // 拒绝原因
}
```
